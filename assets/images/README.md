## Логотип в шапке

Чтобы в шапке сайта отображался ваш логотип KALIPSO (капля + текст), переименуйте файл:

**`ChatGPT Image 7 февр. 2026 г., 12_22_51.png`** → **`logo.png`**

После переименования логотип подставится в шапку автоматически.
