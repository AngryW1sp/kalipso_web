KALIPSO demo (BWT-shop-like)

Файлы:
- index.html — витрина "как интернет-магазин": шапка с категориями + поиск "Я ищу", избранное/корзина, фильтры слева, карточки с ценой/выгодой/в корзину, квиз и переход в Telegram.
- thanks.html — страница "Спасибо".

Подстановка данных через URL:
index.html?city=Москва&phone=%2B7%20(999)%20000-00-00&email=sales%40example.com&b2b=%2B7%20(495)%20000-00-00&tg=kalipso_demo

Предвыбор ветки квиза:
&prefill=pool  или  &prefill=spa
